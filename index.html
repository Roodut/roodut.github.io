ruudut → roodut

Multiple AI platforms running side by side, each in its own pane. We looked for a word that already described it. Finnish had one: ruudut — panes, panels, squares. The building blocks of a grid.


We kept the sound, simplified the spelling, and got Roodut — six letters that fit in a terminal, a repo name, and a product that didn't exist yet.

Features

Everything your AIs need to ship autonomously

Not another chatbot wrapper. Roodut is a full orchestration engine with validation, trust, routing, and overnight execution built in.

⚖

Cross-Platform Validation

One AI builds, another validates. Quality gates enforce minimum scores. Failed validation auto-retries the builder — not the reviewer. Three weaker models outperform GPT-4 when they check each other.

⚕

Trust-Calibrated Autonomy

Agents earn freedom through track record. Start supervised, auto-promote to full autonomy as they prove reliable. Per-task-type permissions — full auto for tests, supervised for production code.

☾

Overnight Execution

Episodic pipeline with bounded episodes, structured handoffs, and git-verified progress. Wake up to completed work, not context cliff disasters. All six death spirals mitigated.

⚙

ENIGMA Routing

Empirical capability matrix scores each agent across task types. Auto-routes tasks to the best-suited agent. Reroute suggestions when quality drops. Stop wasting 80% of AI spend on wrong-model routing.

◉

Real-Time Observability

Watch agents type in real-time — not trace logs after the fact. SSE streaming, live cost tracking, budget caps with auto-halt, and a morning summary digest when you arrive.

▣

Hallucination Detection

Compares what the agent claims it did against the actual git diff. Catches phantom changes, missed files, and fabricated completions before they reach your codebase.

How it works

Install to overnight PRs in minutes

No Python. No graph theory. No YAML configs. Just open Roodut and go.

1

Install

One download. No account, no cloud. It runs entirely on your machine. Your code and prompts never leave your environment.

2

Add your AI platforms

Connect Claude, Codex, Cursor, and Gemini in one place. Each runs natively with its own CLI and credentials — Roodut orchestrates them in parallel terminal panes.

3

Choose from 19+ templates or build your own. Define multi-step workflows — Codex codes, Cursor reviews, Claude summarizes. Set quality gates, budget caps, and oversight level. One run, multiple platforms, one result.

4

Calibrate trust

Set autonomy levels per platform. Use the escalation queue for anything that needs a human. Approve from your phone. Ship faster without losing control.
